# Contributing to DeepSpec

We welcome contributions to add speculative decoding algorithms, improve training performance, or fix bugs!

## Setup Guide

1. Fork and clone the repository.
2. Install the package in editable mode:
   ```bash
   pip install -e ".[dev]"
   ```

## Coding Standards
We use `ruff` to enforce styling and linting:
- To auto-format your changes: `ruff format`
- To check lint errors: `ruff check`

Ensure all checks pass before submitting a Pull Request.
